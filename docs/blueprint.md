# **App Name**: PoemPic

## Core Features:

- AI Poem Generation: Generate a poem based on the content of a user-uploaded photo using generative AI. Use a tool that can reason about visual and semantic information present in the image to formulate a poem. Make creative decisions to choose poem type and overall theme.
- Photo Upload: Allow users to upload a photo from their device.
- Display Poem and Photo: Display the uploaded photo alongside the generated poem.
- Poem Copy: Enable users to easily copy the generated poem to their clipboard.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a vintage, poetic feel.
- Secondary color: Light grey (#D3D3D3) for text and borders.
- Accent: Deep purple (#4B0082) for interactive elements and highlights.
- Serif fonts for headings, sans-serif fonts for body text.
- Simple, line-based icons for a clean aesthetic.
- Clean, minimalistic layout with generous spacing.
- Gentle fade-in animations for the poem and photo to create a smooth user experience.
